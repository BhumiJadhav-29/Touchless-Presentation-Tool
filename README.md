# Touchless Presentation Tool 🎤✋

A Python-based AI project that allows users to control presentations and draw on screen using hand gestures.

## Features
- ✏️ Draw using finger gestures
- 👉 Slide navigation (next/previous)
- ✌️ Pointer mode
- 🧹 Clear screen with fist gesture
- 📷 Real-time hand tracking using MediaPipe

## Tech Used
- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run
1. Install dependencies:
pip install opencv-python mediapipe numpy pynput

2. Run:
python presentation_tool.py
